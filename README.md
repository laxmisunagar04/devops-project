## 📌 Project Overview
This project demonstrates a complete DevOps CI/CD pipeline using GitHub Actions, Selenium testing, and automated deployment concepts.

It runs automated tests on every code push and ensures code quality before deployment.

## 🛠️ Tech Stack / Tools Used

- Git & GitHub for version control  
- GitHub Actions for CI/CD automation  
- Selenium for browser automation testing  
- PyTest for test execution  
- HTML/CSS for frontend  


## ⚙️ Workflow

1. Developer pushes code to GitHub  
2. GitHub Actions triggers automatically  
3. PyTest + Selenium tests run  
4. If tests pass → pipeline succeeds
